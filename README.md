## Power BI Project 3: UPI Transactions Data Analysis

This project focuses on analyzing UPI transaction data using Power BI with a strong emphasis on interactivity and user experience.

---

### 📊 Objective
To analyze transaction patterns, monitor trends, and create an interactive dashboard for better financial insights.

---

### ⚙️ Data Preparation
- Loaded dataset into Power BI Desktop
- Performed data profiling and validation
- Created derived columns such as Age Group
- Structured data for reporting

---

### 📈 Visualizations
- Line Chart for transaction trends
- Matrix Visual for detailed analysis
- Interactive slicers for filtering

---

### 🔍 Features
- Slicer customization (size, position, formatting)
- Sync slicers across report pages
- Conditional formatting for better insights
- Bookmark-based navigation

---

### 🎯 Interactivity
- Bookmarks to switch between:
  - Transactions view
  - Remaining Balance view
- Dynamic filtering using slicers

---

### 🚀 Deployment
- Published report to Power BI Service
- Enabled sharing and accessibility

---

### ✅ Outcome
- Improved visibility into transaction behavior
- Enhanced user experience with interactive dashboard design

---

### 🛠 Tools Used
- Power BI Desktop
- Power BI Service
- Power Query
